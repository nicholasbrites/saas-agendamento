# 📅 SaaS Agendamento

Projeto em desenvolvimento com o objetivo de criar uma plataforma SaaS para gerenciamento de agendamentos (ex: barbearias).

## 🚀 Sobre o projeto

A proposta é desenvolver um sistema onde usuários possam:
- Visualizar estabelecimentos
- Realizar login
- Agendar horários com profissionais
- Gerenciar seus agendamentos

## 📚 Base de estudo

Este projeto está sendo desenvolvido com base em um curso prático de desenvolvimento full stack, abordando:

1. Node.js  
2. React  
3. TypeScript  
4. Next.js  
5. Setup do projeto  
6. Autenticação (Google)  
7. Sistema de agendamentos  
8. Integração com pagamentos  
9. Deploy da aplicação
